Studio Team Assignment - Doppler Effect Simulation

We have used react.js to build the assignment and for UI screens we have used HTML,CSS(not used scss for this) and Bootstrap.


How to Setup Code?

Extract the assignment zip and open the command prompt or terminal in same project directory and run 'yarn install' command.


How to run the application : 

Open the terminal or command prompt in project directory and run 'yarn start' command.
